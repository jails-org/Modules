[Redux](http://redux.js.org/)


## Methods

### new Reduxtore( reducer, initialState )

### .dispatch( action )
Dispatch an action to the store.

### .getState()
Gets the initial state of the store.

### .subscribe()
Subscribe a listener that will be executed after dispatching an action.
